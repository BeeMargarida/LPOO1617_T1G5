Detailed Description

CrazyOgre is a class that keeps information about the enemy CrazyOgre of the game. This enemy has a symbol, coordinates x and y, two booleans representing if he is in the same position of the key and if it is stunned (was it by the hero weapon).It also keeps information about the turns it still has to wait while stunned and it's weapon.

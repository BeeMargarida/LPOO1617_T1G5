Detailed Description

Game is a class that deals with the whole game, connecting logics with the respective map. This class checks if the game is over and either proceeds to the next level (if there is one), or ends the game and transmits that information to the main function. It contains the map and logic of the game, the enemy's options of each level and the current level of the game being played.

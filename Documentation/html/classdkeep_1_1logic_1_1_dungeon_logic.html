Detailed Description

DungeonLogic is a class that extends Logic and it's function is to deal with the logic of the Dungeon level. It contains all the specific functions that deal with the hero and the type of enemies of its level. The big difference in this class, compared to the other, is the existence of three guards, each with a different behaviour. This level only has a lever.
